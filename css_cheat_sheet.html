CSS Cheat Sheet

A CSS cheat sheet is a quick reference guide
that provides a summary of the most commonly used CSS properties and their syntax.
It is useful for beginners and experts alike who need a quick reference to help them learn and remember the syntax of CSS.

1. Basic Syntax & Selectors

CSS Rule Structure:

selector {
  property: value;
  /* More properties... */
}

Comment:

/* This is a comment */
Selector Type Syntax Example Description
Universal * { margin: 0; } Targets all elements
Type (Element) p { font-size: 16px; } Targets all <p> elements
Class .btn { background: blue; } Targets all elements with class="btn"
ID #header { padding: 20px; } Targets the element with id="header"
Descendant div p { color: red; } Targets all <p> elements inside any <div>
Child ul > li { list-style: none; } Targets direct <li> children of a <ul>
Adjacent Sibling h1 + p { margin-top: 0; } Targets the first <p> immediately after an <h1>
General Sibling h1 ~ p { color: gray; } Targets all <p> siblings following an <h1>
Attribute a[target="_blank"] { color: green; } Targets <a> elements with target="_blank"
Pseudo-class a:hover { text-decoration: underline; } Applies styles when the element is in a specific state (e.g., hover, focus)
Pseudo-element p::first-line { font-weight: bold; } Targets a specific part of an element (e.g., the first line)

2. The Box Model

Property Syntax Example Description
Width / Height width: 300px; height: 200px; Sets the width and height of an element
Padding padding: 10px; Space between the content and the border
Border border: 1px solid #333; Sets the border’s width, style, and color
Margin margin: 15px; Space outside the border
Box Sizing box-sizing: border-box; Includes padding and border in the element’s total width and height

3. Positioning

Property Syntax Example Description
Static position: static; Default; element follows normal document flow
Relative position: relative; top: 10px; Offsets the element relative to its normal position
Absolute position: absolute; top: 0; left: 0; Removes the element from normal flow; positioned relative to its nearest positioned ancestor
Fixed position: fixed; bottom: 0; Fixed relative to the viewport
Sticky position: sticky; top: 0; Behaves like relative until a given offset is met, then sticks in place
Z-index z-index: 10; Sets the stacking order for overlapping elements

4. Layout Techniques

A. Flexbox

Property Syntax / Example Description
Enable Flex display: flex; Declares a flex container
Flex Direction flex-direction: row; or column; Sets the direction of flex items
Justify Content justify-content: center; Aligns items along the main axis
Align Items align-items: center; Aligns items along the cross axis
Flex Wrap flex-wrap: wrap; Allows items to wrap onto multiple lines
Flex Grow flex-grow: 1; Determines how much a flex item will grow relative to others
Flex Shrink flex-shrink: 1; Determines how much a flex item will shrink relative to others
Flex Basis flex-basis: 200px; Sets the initial main size of a flex item
Order order: 2; Changes the visual order of flex items (default is 0)

B. CSS Grid

Property Syntax / Example Description
Enable Grid display: grid; Declares a grid container
Grid Template Columns grid-template-columns: 1fr 2fr; Defines the columns (using fractions, pixels, percentages, etc.)
Grid Template Rows grid-template-rows: auto 100px; Defines the rows
Gap (Grid Gap) gap: 20px; or grid-gap: 20px; Sets spacing between grid items
Grid Column grid-column: 1 / span 2; Specifies where a grid item starts and how many columns it spans
Grid Row grid-row: 1 / 3; Specifies where a grid item starts and ends in rows
Grid Template Areas 
@grid-template-areas:
"header header"
"sidebar main"
"footer footer";
 Defines a layout by naming areas for grid items
Align Items / Justify Items align-items: center;
justify-items: center;		 Controls the alignment of items within grid areas

5. Responsive Design

Media Queries

Syntax Example Description
Basic Media Query 
@media (max-width: 600px) {
  /* CSS rules for small screens */
}
 Applies styles when the viewport width is 600px or less
Min-Width Query (Mobile-First) 
@media (min-width: 768px) {
  /* CSS rules for tablets and up */
}
 Applies styles for viewports 768px and wider
Orientation Query 
@media (orientation: portrait) {
  /* CSS rules for portrait mode */
}
 Applies styles when the device is in portrait mode
High-Resolution (Retina) Displays 
@media (-webkit-min-device-pixel-ratio: 2), (min-resolution: 192dpi) {
  /* High-res styles */
}
 Targets devices with high pixel density

Note: Include the following meta tag in your HTML to ensure proper responsive behavior:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

6. Additional CSS Features

Transitions

Property Syntax / Example Description
Transition transition: background-color 0.3s ease; Animates property changes over time

Animations

Rule/Property Syntax / Example Description
@keyframes 
@keyframes slide {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}
 Defines the steps of an animation
Animation animation: slide 0.5s linear; Applies the defined animation to an element

Transforms

Property Syntax / Example Description
Transform transform: rotate(45deg) scale(1.5) translateX(20px); Applies geometric transformations (rotation, scaling, translation)
Transform Origin transform-origin: center; Sets the point around which a transformation is applied

CSS Variables (Custom Properties)

Syntax Example Description
Define a Variable 
:root {
  --main-color: #3498db;
}
 Declares a global CSS variable
Use the Variable color: var(--main-color); Retrieves the value of the custom property
