HTML Cheat Sheet

HTML (Hypertext Markup Language) is the standard markup language for creating web pages and web applications.
It defines the structure and presentation of content on the web pages. It is the core building block of the web.

1. Basic Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Page Title</title>
  <!-- Additional metadata, links, or scripts go here -->
</head>
<body>
  <!-- Your page content goes here -->
</body>
</html>

Description: This is the HTML5 boilerplate. The <!DOCTYPE html> declaration triggers standards mode. The <meta> tags ensure proper character encoding and responsive behavior.

2. Common HTML Elements

Element Syntax/Example Description
Heading <h1>Main Heading</h1> Use <h1> to <h6> for headings; <h1> is the highest level.
Paragraph <p>This is a paragraph of text.</p> Represents a block of text.
Link <a href="https://example.com">Visit Example</a> Creates a hyperlink to another resource.
Image <img src="image.jpg" alt="Description"> Embeds an image; always include an alt attribute.
Line Break <br> Inserts a line break.
Horizontal Rule <hr> Inserts a thematic break (a horizontal line).
Comment <!-- This is a comment --> Comments are not rendered by the browser.

3. Text Formatting & Semantic Elements

Element Syntax/Example Description
Bold (Strong) <strong>Important</strong> or <b>Bold Text</b> <strong> indicates importance; <b> is for styling only.
Italic (Emphasis) <em>Emphasized</em> or <i>Italic Text</i> <em> provides emphasis; <i> is typically used for stylistic changes.
Underline <u>Underlined Text</u> Underlines text (use sparingly as it may imply links).
Subscript / Superscript <sub>subscript</sub> / <sup>superscript</sup> Renders text as subscript or superscript.
Mark <mark>Highlighted</mark> Highlights text (often used to denote search results).
Code <code>inline code</code> Marks inline code snippets.
Preformatted <pre>Preformatted text</pre> Preserves whitespace and line breaks (ideal for code blocks or poems).

4. Lists

List Type Example Description
Unordered List 
<ul>
  <li>Item One</li>
  <li>Item Two</li>
  <li>Item Three</li>
</ul>
Creates a bullet-point list.
Ordered List 
<ol>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ol>
Creates a numbered list.
Description List 
<dl>
  <dt>Term</dt>
  <dd>Description of the term.</dd>
</dl>
Defines a description list (term-description pairs).

5. Tables

Example Description 
<table>
  <caption>Table Caption</caption>
  <thead>
    <tr>
      <th>Header 1</th>
      <th>Header 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Data 1</td>
      <td>Data 2</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Footer 1</td>
      <td>Footer 2</td>
    </tr>
  </tfoot>
</table>
Provides a full table structure with a caption, header, body, and footer.

6. Forms & Input Elements

Element Syntax/Example Description
Form <form action="/submit" method="post"> ... </form> Wraps form elements; action specifies where to send data; method defines the HTTP method.
Text Input <input type="text" name="username" placeholder="Enter username"> Single-line text input.
Password Input <input type="password" name="password"> Input field for passwords (characters are hidden).
Email Input <input type="email" name="email"> Input field with built-in email validation.
Submit Button <button type="submit">Submit</button> or <input type="submit" value="Submit"> Triggers form submission.
Checkbox <input type="checkbox" name="subscribe" checked> Represents a binary choice; the checked attribute pre-selects the box.
Radio Button <input type="radio" name="gender" value="male"> Male Allows selection of one option from a set.
Select Dropdown 
<select name="options">
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
</select>
Creates a dropdown list.
Textarea <textarea name="message" rows="4" cols="50"></textarea> Multi-line text input area.
Label <label for="username">Username:</label> <input id="username" type="text" name="username"> Improves accessibility by associating text with form controls.

7. Media Elements

Element Syntax/Example Description
Image <img src="image.jpg" alt="Description"> Embeds an image; include descriptive alt text.
Audio 
<audio controls src="audio.mp3">
  Your browser does not support the audio element.
</audio>
Embeds audio with built-in controls.
Video 
<video controls width="320" height="240">
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>
Embeds video content; can include multiple sources.
Embed <embed src="file.pdf" width="500" height="375" type="application/pdf"> Embeds external content like PDFs or Flash content.
Object <object data="file.swf" type="application/x-shockwave-flash" width="400" height="300"></object> Embeds multimedia or interactive content.

8. Semantic Elements

Element Syntax/Example Description
Header <header> ... </header> Typically contains introductory content or navigation.
Nav <nav> ... </nav> Contains site navigation links.
Main <main> ... </main> Represents the primary content of the document.
Article <article> ... </article> Encapsulates independent, self-contained content (e.g., a blog post).
Section <section> ... </section> Groups related content thematically.
Aside <aside> ... </aside> Contains tangentially related content (like sidebars).
Footer <footer> ... </footer> Contains footer information for a document or section.
Figure & Figcaption 
<figure>
  <img src="img.jpg" alt="Image Description">
  <figcaption>Image caption</figcaption>
</figure>
Groups media with its caption.

9. Meta Tags & Linking Resources

Element Syntax/Example Description
Meta Charset <meta charset="UTF-8"> Defines the character encoding.
Meta Viewport <meta name="viewport" content="width=device-width, initial-scale=1.0"> Ensures proper scaling on mobile devices.
Meta Description <meta name="description" content="A brief description of the page"> Helps search engines understand the page content.
Link CSS <link rel="stylesheet" href="styles.css"> Links an external stylesheet.
Script <script src="script.js"></script> Links an external JavaScript file.

10. Scripting & Data

Element Syntax/Example Description
Inline Script <script>console.log('Hello, world!');</script> Embeds JavaScript directly within the HTML.
Data Attributes <div data-user="JohnDoe"></div> Custom attributes (prefixed with data-) to store extra data.

11. Accessibility & ARIA

Attribute/Element Syntax/Example Description
ARIA Role <div role="navigation"> ... </div> Helps define the purpose of an element for assistive technologies.
ARIA Label <button aria-label="Close">X</button> Provides an accessible label where visible text is insufficient.
Tabindex <a href="#" tabindex="0">Focusable Link</a> Makes an element focusable via keyboard navigation.

12. Miscellaneous Elements

Element Syntax/Example Description
Iframe <iframe src="https://example.com" title="Example Site" width="600" height="400"></iframe> Embeds another HTML document within the current page.
Base <base href="https://example.com/"> Sets a base URL for all relative URLs in the document.
Noscript <noscript>Your browser does not support JavaScript!</noscript> Provides fallback content for users with JavaScript disabled.

This cheat sheet covers the core HTML elements and patterns you’ll use day-to-day. For more in-depth learning, refer to the official MDN Web Docs or other comprehensive resources.

Happy coding!
