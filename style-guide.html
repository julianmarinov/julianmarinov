Colors

The website uses a color palette with a light and dark theme. The light theme uses a light background color and dark text color, while the dark theme uses a dark background color and light text color. The colors used are:

  • Light Theme:
    • #FFFFFF
    (white)
  • Dark Theme:
    • #000000
    (black)

Buttons

Buttons are used throughout the website to provide a consistent user experience. The buttons have a flat design and a rounded border to make them stand out. The buttons are styled using CSS and are in light and dark theme.
