WebStorm Cheat Sheet for MacOS

WebStorm is a powerful IDE for front-end web development, built by JetBrains.
This cheat sheet provides a quick reference for the most common actions and shortcuts in WebStorm on MacOS.

Editing & Code Manipulation

Action Shortcut (Mac) Description / Notes
Comment/Uncomment Line ⌘ + / Toggle a single-line comment on the current line
Block Comment/Uncomment ⌘ + Shift + / Wrap (or unwrap) the selected code in a block comment
Duplicate Line/Selection ⌘ + D Duplicate the current line or selected block
Delete Line ⌘ + Delete (or ⌘ + Backspace) Delete the entire current line
Surround With... ⌘ + Alt + T Wrap the selection in a code construct (e.g., if/try-catch)
Code Completion (Basic) Control + Space Invoke basic code completion suggestions
Code Completion (Smart) Control + Shift + Space Trigger context-aware (smart) completions
Reformat Code ⌘ + Alt + L Auto-format code according to your style settings
Multiple Cursors / Carets Option + Click Add additional carets for simultaneous editing

Navigation

Action Shortcut (Mac) Description / Notes
Go to Declaration/Definition ⌘ + B or ⌘ + Click Jump to the declaration/definition of a symbol
Go to Implementation ⌘ + Alt + B Navigate to a symbol’s implementation
Navigate Back / Forward ⌘ + [ / ⌘ + ] Move backward or forward through your navigation history
Open File ⌘ + O Open a file by name
Recent Files ⌘ + E Show a list of recently opened files
Search Everywhere Shift (double-tap) Search for files, symbols, actions, and settings
Go to Symbol ⌘ + Alt + O Quickly jump to any symbol in your project

Refactoring

Action Shortcut (Mac) Description / Notes
Rename Shift + F6 Rename a symbol and update all its usages
Extract Variable ⌘ + Alt + V Extract an expression into a new variable
Extract Method/Function ⌘ + Alt + M Extract a block of code into its own method/function

Search & Replace

Action Shortcut (Mac) Description / Notes
Find in File ⌘ + F Open search within the current file
Replace in File ⌘ + R Replace text in the current file
Find in Path (Project-wide) ⌘ + Shift + F Search across the entire project
Replace in Path ⌘ + Shift + R Replace text across all files in your project
Find Action ⌘ + Shift + A Search for IDE actions, settings, or commands

Code Generation & Emmet

Action Shortcut (Mac) Description / Notes
Emmet Expand Abbreviation Type abbreviation + Tab Type an Emmet abbreviation (e.g., type ! for HTML boilerplate) and press Tab to expand
Generate Code ⌘ + N Open the generate menu (useful for generating code snippets like constructors, etc.)

Running & Debugging

Action Shortcut (Mac) Description / Notes
Run Control + R Run the current configuration (if set up)
Debug Control + D Start debugging the current configuration

Miscellaneous

Action Shortcut (Mac) Description / Notes
Show Intention Actions (Quick Fix) Alt + Enter Display available quick fixes and intention actions
Quick Documentation Lookup F1 (or Fn + F1) View documentation for the symbol at the caret

Tips

  • Emmet is built into WebStorm for HTML, CSS, and more. Just type an abbreviation (like !) and press Tab to expand it.
  • Use ⌘ + Shift + A to search for any action by name—this is great if you forget a shortcut.
  • Many shortcuts are similar across JetBrains IDEs, so if you switch between them, your muscle memory will soon adjust.

Happy coding!
